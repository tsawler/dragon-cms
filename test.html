Drag & Drop Editor Test

To test the editor:

  1. Open index.html in your browser
  2. Drag a "Container Block" from the left panel to the main area
  3. Then drag snippets (like "Heading" or "Paragraph") into the block
  4. Click on text to edit it directly
  5. Use the icons on hover to:
    • ✏️ Edit styles
    • </> Edit raw HTML
    • 🗑️ Delete element
  6. Try the undo/redo buttons
  7. Switch between Edit and Display modes
  8. Save your work and export as HTML
